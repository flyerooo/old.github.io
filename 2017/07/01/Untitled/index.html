  • JavaScript入门经典
    • HTML表单
      • 主要内容
        • 给文本、密码、文本区、和隐藏的表单控件编写脚本
        • 为选择、复选框、单选按钮表单控件编写代码
        • 使用JavaScript与新的HTML5表单控件交互
      • 表单共有的属性和方法
        • name属性
        • value属性
        • form属性
        • type属性
        • focus()和blur()方法
          • onfocus(),onblur()
      • 按钮元素
        • button
        • submit
        • reset
      • 文本元素
        • 文本框
          • select()
          • 最好显示转换数据类型
        • Firefox和blur时间的问题
        • 密码文本框
        • 隐藏文本框
        • textarea
    • 事件
      • 事件的类型
        • 鼠标事件
        • 键盘事件
        • 进度事件
        • 表单事件
        • 突变事件
          • 修改DOM节点时发生
        • 触摸事件
        • 错误事件
      • 将代码连接到事件
        • 通过HTML属性
          • 缺点
            • HTML和JavaScript混合在一起，因此很难维护，查找，修改错误
            • 不修改HTML,就不能修改事件处理程序
            • 只能为HTML代码中出现的元素建立事件处理程序，不能为动态创建的元素建立事件处理程序
        • 通过对象的特定属性
        • 通过调用对象的特定方法
      • 标准事件模型
        • 优点
          • 可以为一个元素的一个事件注册多个事件监听器
      • 旧版本IE中的事件处理
        • 使用全局的event对象
        • 不需要像DOM事件模型那样，把该对象传送给处理函数
        • 与标准事件模型中addEventListener()和removeEventListener()类似的方法：attachEvent()和detachEvent()
          • 但在使用时，事件名有前缀“on”
      • 编写跨浏览器的代码
        • 特性检测
        • 浏览器嗅探
      • 内置拖放操作
        • 拖动源事件
          • dragstart
            • 要执行拖放操作，唯一需要监听的事件是dragstart
          • drag
          • dragend
        • 创建释放目标
        • 传输数据
          • 拖放操作就是传递数据
            • 开始操作时，需要告诉浏览器我们打算传递什么数据，
            • 释放对象时，需要指定如何把数据从源传递到目的地
            • dataTransfer对象用于存储在拖放操作中被拖动的数据

如何写好原生Javascript?

看了齐舞团“如何写好原生Javascript?”，发现有些知识点掌握的还是不太牢。再次记录一下。

  • 代码职责清晰，JavaScript不应该直接操作DOM
  • API设计合理
  • 合适的算法
    例. 给定一个很大的数组，里面有很多的整数，有JavaScript实现一个函数，要求将数组中和为10的每一对数配对并找出，返回这些数配对后的数组。

先排序，首尾相加，如和等于10，两头都向中间靠拢，和小于10，低位数加1，和大于19，高位数减1
